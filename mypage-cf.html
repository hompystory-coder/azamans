1 시작하기 - API 키 설정

쿠팡 파트너스 API 키를 등록하세요:

  1. 쿠팡 파트너스 가입
    • 쿠팡 파트너스 사이트 접속
    • 회원가입 및 승인 대기
  2. API 키 발급
    • 쿠팡 파트너스 로그인
    • 도구 → API → Access Key, Secret Key 복사
  3. 마이페이지에 등록
    • "API 키 및 설정" 섹션에서 Access Key, Secret Key 입력
    • "API 키 저장" 버튼 클릭

중요: Secret Key는 암호화되어 저장되며, 한 번 저장하면 마스킹(••••)으로 표시됩니다.

2 YouTube 채널 연동

내 YouTube 채널을 연결하세요:

  1. 채널 URL 또는 ID 입력
    • 채널 ID: UCxxxxxxxxxxxxxxxxxx
    • 채널 핸들: @채널이름
    • 전체 URL: https://www.youtube.com/@채널이름
  2. 자동 파싱
    • URL을 붙여넣으면 자동으로 채널 ID 추출
    • 입력 필드에서 포커스를 벗어나면 자동 변환
  3. 저장
    • "API 키 저장" 버튼으로 함께 저장

3 쇼츠 등록하기 (3가지 방법)

방법 1: 쇼츠 불러오기 (권장)

채널에서 최신 쇼츠 10개를 한 번에 가져옵니다.

  1. "쇼츠 불러오기" 버튼 클릭
  2. 자동으로 제목, 설명, 조회수, 좋아요 수집
  3. 이미 등록된 쇼츠는 자동으로 제외
  4. 한글 설명 자동 수집

방법 2: 개별 등록

특정 쇼츠 URL을 직접 입력하여 등록합니다.

  1. "쇼츠 추가" 버튼 클릭
  2. YouTube 쇼츠 URL 입력 (예: https://youtube.com/shorts/xxxxx)
  3. 자동으로 정보 수집
  4. 필요시 상품 링크도 함께 입력
  5. "등록" 버튼 클릭

방법 3: 자동 수집 (자동화)

설정한 시간마다 자동으로 새 쇼츠를 수집합니다.

  1. "자동 쇼츠 수집" 섹션으로 이동
  2. "자동 수집 활성화" 토글 ON
  3. 수집 간격 설정 (예: 24시간)
  4. "자동 수집 설정 저장" 클릭
  5. 이후 자동으로 새 쇼츠만 수집됨

4 자동 쇼츠 수집 설정

한 번 설정하면 자동으로 쇼츠를 수집합니다:

✨ 자동 수집의 장점

  • 매번 수동으로 등록할 필요 없음
  • 새 영상 업로드 시 자동 감지
  • 이미 등록된 것은 건너뜀 (중복 방지)
  • 시간 절약 및 효율성 증대

설정 방법:

  1. 활성화: 토글 스위치를 ON으로 전환
  2. 간격 설정: 1~168시간 (1시간~7일) 중 선택
    • 매일 업로드: 6-12시간 권장
    • 주 2-3회: 24시간 권장
    • 주 1회: 48-72시간 권장
  3. 즉시 테스트: "지금 실행" 버튼으로 바로 테스트
  4. 저장: "자동 수집 설정 저장" 클릭

5 쇼츠 관리하기

등록된 쇼츠를 효율적으로 관리하세요:

개별 관리

  • 수정: 쇼츠 정보 수정 (제목, 설명, 상품 링크 등)
  • 삭제: 쇼츠 삭제

일괄 관리 (시간 절약)

  1. 선택: 체크박스로 여러 쇼츠 선택
  2. 전체 선택: "전체 선택" 체크박스 클릭
  3. 일괄 작업:
    • "대기 상태로 변경" - 승인/반려된 것을 다시 대기로
    • "선택 항목 삭제" - 여러 개 한 번에 삭제

팁: 일괄 삭제는 실수로 삭제할 수 있으니 신중하게 선택하세요!

6 승인 대기 및 수익 확인

관리자 승인 후 수익 발생:

📊 대시보드 통계

  • 총 쇼츠: 등록된 모든 쇼츠 개수
  • 총 클릭: 사용자가 상품 링크를 클릭한 횟수
  • 총 수익: 쿠팡 파트너스 수익 합계
  • 승인 대기: 관리자 승인 대기 중인 쇼츠

🔄 승인 프로세스

  1. 등록: 쇼츠 등록 → 자동으로 "승인 대기" 상태
  2. 관리자 검토: 관리자가 내용 확인
  3. 승인: 문제없으면 승인 → 홈 페이지에 표시
  4. 반려: 문제 있으면 반려 (사유 메시지)

승인 후: 홈 페이지에 노출되어 사용자가 볼 수 있습니다.

자주 묻는 질문 (FAQ)

Q: 쇼츠 등록 후 언제 승인되나요?

A: 관리자가 검토 후 승인합니다. 보통 1-3일 소요됩니다.

Q: 자동 수집이 작동하지 않아요.

A: YouTube 채널 ID가 올바르게 입력되었는지, 자동 수집이 활성화되었는지 확인하세요.

Q: 삭제한 쇼츠를 복구할 수 있나요?

A: 삭제 후에는 복구할 수 없으니 신중하게 삭제하세요.

Q: 쇼츠 설명이 영어로 나와요.

A: 최신 버전에서는 자동으로 한글 수집됩니다. 기존 영어 설명은 다시 불러오기 하세요.

Q: 쿠팡 API 키는 어디서 발급받나요?

A: 쿠팡 파트너스 사이트 → 로그인 → 도구 → API에서 발급받을 수 있습니다.

추가 문의사항이 있으시면 관리자에게 문의해주세요.
