## 🌟 전체 캐릭터 라인업 (10가지)

### 기존 캐릭터 (3개)

#### 1. 🦊 똑똑한 여우 (clever-fox)
- **성격:** 똑똑하고 재치있는
- **적합 콘텐츠:** 비즈니스, 재테크, 뉴스, 정보
- **스타일:** 친절하고 전문적인 제품 설명

#### 2. 🐰 행복한 토끼 (happy-rabbit)
- **성격:** 밝고 발랄한
- **적합 콘텐츠:** 라이프스타일, 뷰티, 패션, 일상
- **스타일:** 즐겁고 활기찬 제품 소개
- **인사:** "안녕! 행복한 토끼가 [제품]를 보여줄게요!"

#### 3. 🦉 현명한 부엉이 (wise-owl)

- **적합 콘텐츠:** 교육, 학습, 자기계발, 독서
- **스타일:** 전문적이고 신뢰감 있는 분석

## 📝 사용 방법

### 웹 인터페이스
1. https://mfx.neuralgrid.kr/crawler 접속
2. 블로그 URL 입력 및 크롤링
3. https://mfx.neuralgrid.kr/character 에서 **10가지 캐릭터 중 선택**
4. 영상 생성 시작!

### API 사용
```bash
curl -X POST https://mfx.neuralgrid.kr/api/character-shorts \
  -H "Content-Type: application/json" \
  -d '{
    "blogUrl": "https://blog.naver.com/alphahome/224106828152",
    "characterId": "creative-penguin"
  }'
```
