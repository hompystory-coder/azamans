### 1️⃣ **블로그 기사 쇼츠생성기** (Blog Shorts Generator)
**🔗 URL:** https://bn-shop.neuralgrid.kr

#### 📝 설명
블로그 기사를 AI가 자동으로 분석하여 YouTube Shorts 형식의 영상으로 변환. Gemini 2.0, Pollinations.AI, Kling v2.1 Pro를 활용한 완전 자동화 솔루션.

#### ✨ 주요 기능
- 📄 블로그 URL 입력 → 자동 콘텐츠 분석
- 🎬 AI 스토리보드 자동 생성 (Gemini 2.0)
- 🖼️ 이미지 생성 (Pollinations.AI)
- 🎥 비디오 렌더링 (Kling v2.1 Pro)
- 🔊 한국어 자막 자동 렌더링
- ⚡ 평균 처리 시간: 4~5분

#### 📖 사용 방법
1. bn-shop.neuralgrid.kr 접속
2. "새 프로젝트 만들기" 클릭
3. 블로그 URL 입력
4. AI 자동 분석 시작 (4-5분 소요)
5. 완성된 Shorts 영상 다운로드

#### 💰 가격
영상당 약 $0.06

---

### 2️⃣ **쇼츠 영상 자동화** (MediaFX Shorts)
**🔗 URL:** https://mfx.neuralgrid.kr

#### 📝 설명
AI 기반 숏폼 비디오 자동 생성 플랫폼. 텍스트 프롬프트만으로 전문가급 Shorts 영상을 3-4분 만에 제작.

#### ✨ 주요 기능
- 🤖 텍스트 → 비디오 자동 생성
- 🎨 다양한 템플릿 및 스타일 지원
- 🎵 AI 음악 자동 매칭
- 📱 SNS 최적화 (Instagram, TikTok, YouTube)
- 🔄 배치 프로세싱 지원
- 📊 영상 성과 분석 대시보드

#### 📖 사용 방법
1. mfx.neuralgrid.kr 접속 → SSO 로그인
2. "새 쇼츠 만들기" 클릭
3. 텍스트 프롬프트 입력 (예: "도쿄 야경 감성 브이로그")
4. 스타일 및 음악 선택
5. "생성하기" 클릭 → 3-4분 대기
6. 완성된 영상 다운로드 또는 직접 SNS 업로드

#### 💰 가격
유료 (경쟁력 있는 가격)

### 3️⃣ **스타뮤직** (NeuronStar Music)
**🔗 URL:** https://music.neuralgrid.kr

#### 📝 설명
무료 AI 음악 생성 플랫폼. 텍스트 설명만으로 고품질 음악을 30-60초 만에 제작하고, 상업적으로 자유롭게 활용.

#### ✨ 주요 기능
- 🎼 다양한 장르 지원 (Pop, Rock, Jazz, EDM, Classical)
- ✍️ 커스텀 가사 입력 가능
- 🎧 고품질 오디오 출력 (WAV/MP3)
- 💼 상업적 이용 가능
- 📦 무제한 생성 (무료 플랜)
- 🔊 실시간 미리듣기

#### 📖 사용 방법
1. music.neuralgrid.kr 접속 → SSO 로그인
2. "새 음악 만들기" 클릭
3. 음악 설명 입력 (예: "잔잔한 피아노 선율의 감성 발라드")
4. 장르 및 분위기 선택
5. (선택) 가사 입력
6. "생성하기" 클릭 → 30-60초 대기
7. 완성된 음악 청취 후 다운로드 (MP3/WAV)

#### 💰 가격
완전 무료 (Free Forever)

### 4️⃣ **쿠팡쇼츠** (Shorts Market)
**🔗 URL:** https://market.neuralgrid.kr

#### 📝 설명
YouTube Shorts와 쿠팡 파트너스를 연동한 쇼츠 커머스 플랫폼. 자동 쇼츠 수집, 딥링크 생성으로 크리에이터 수익을 극대화.

#### ✨ 주요 기능
- 📹 YouTube Shorts 자동 수집 (100% 성공률)
- 🛒 쿠팡 파트너스 딥링크 자동 생성
- 🔗 네이버 브랜드 커넥트 지원
- 💰 크리에이터 수익 대시보드
- 📊 조회수/좋아요/댓글 통계
- 🎯 자동 상품 매칭

#### 📖 사용 방법
1. market.neuralgrid.kr 접속 → SSO 로그인
2. "새 쇼츠 등록" 클릭
3. YouTube Shorts URL 입력
4. 자동 정보 수집 (제목, 썸네일, 조회수 등)
5. 쿠팡 상품 링크 입력 또는 자동 추천 선택
6. "등록하기" 클릭
7. 크리에이터 대시보드에서 수익 추적

#### 💰 가격
무료 (베타)

### 5️⃣ **N8N 워크플로우 자동화**
**🔗 URL:** https://n8n.neuralgrid.kr

#### 📝 설명
200개 이상의 앱 연동을 지원하는 강력한 워크플로우 자동화 엔진. 복잡한 비즈니스 프로세스를 드래그 앤 드롭으로 자동화.

#### ✨ 주요 기능
- 🔗 200+ 앱/서비스 통합 지원
- 🖱️ 드래그 앤 드롭 빌더
- 🔄 REST API 자동화
- ⏰ Cron 스케줄링
- 🔐 보안 자체 호스팅
- 📊 실시간 로그 및 디버깅

#### 📖 사용 방법
1. n8n.neuralgrid.kr 접속 → SSO 로그인
2. "새 워크플로우 만들기" 클릭
3. 트리거 노드 선택 (예: Webhook, Schedule)
4. 작업 노드 연결 (예: Send Email, HTTP Request)
5. 각 노드 설정 및 연결
6. "활성화" 버튼 클릭
7. 워크플로우 자동 실행 모니터링

#### 💰 가격
무료 (Self-hosted)

### 6️⃣ **서버모니터링** (System Monitor)
**🔗 URL:** https://monitor.neuralgrid.kr

#### 📝 설명
실시간 서버 상태 및 PM2 프로세스를 한눈에 모니터링하는 프리미엄 대시보드. Glassmorphism 디자인과 Chart.js 그래프 제공.

#### ✨ 주요 기능
- 📊 실시간 CPU/메모리 사용률 (Chart.js)
- 💿 디스크 용량 모니터링
- ⚙️ PM2 프로세스 상태 추적
- 🔔 자동 알림 시스템
- 🎨 Glassmorphism 디자인
- 🔄 30초 자동 갱신

#### 📖 사용 방법
1. monitor.neuralgrid.kr 접속 → SSO 로그인
2. 실시간 대시보드 자동 표시
3. 각 서비스 상태 확인 (🟢 녹색: 정상 / 🔴 빨간색: 중단)
4. CPU/메모리 그래프 실시간 모니터링
5. PM2 프로세스 제어 (재시작/중지)

#### 💰 가격
무료 (Free)

### 7️⃣ **통합 인증 서비스** (Auth Service)
**🔗 URL:** https://auth.neuralgrid.kr

#### 📝 설명
모든 NeuralGrid 서비스를 위한 중앙 집중식 JWT 기반 SSO 인증 시스템. 한 번의 로그인으로 모든 서비스에 즉시 접근.

#### ✨ 주요 기능
- 🔐 JWT 기반 보안 인증
- 🎫 API 키 발급 및 관리
- 💳 크레딧 추적 시스템
- 👤 통합 사용자 관리
- 🔄 자동 토큰 갱신
- 🛡️ 권한 기반 접근 제어

#### 📖 사용 방법
1. 모든 서비스에서 "로그인" 클릭
2. auth.neuralgrid.kr 자동 리다이렉트
3. 이메일/비밀번호 입력
4. JWT 토큰 자동 발급
5. 원래 서비스로 자동 복귀
6. 모든 서비스 즉시 사용 가능

#### 💰 가격
무료 (Free)
