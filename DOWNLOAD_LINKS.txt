📦 1단계: 필수 프로그램 다운로드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Python 3.10 (필수)
   다운로드: https://www.python.org/downloads/windows/
   파일명: python-3.10.11-amd64.exe
   용량: 25 MB
   설치 시간: 5분
   ⚠️  중요: 설치 시 "Add Python to PATH" 반드시 체크!

✅ Git (필수)
   다운로드: https://git-scm.com/download/win
   파일명: Git-2.43.0-64-bit.exe
   용량: 50 MB
   설치 시간: 3분

✅ Ollama (필수 - LLaMA 3.1 실행용)
   다운로드: https://ollama.ai/download/windows
   파일명: OllamaSetup.exe
   용량: 500 MB
   설치 시간: 5분
   용도: AI 스크립트 생성

✅ FFmpeg (필수 - 영상 처리)
   다운로드: https://www.gyan.dev/ffmpeg/builds/
   파일명: ffmpeg-release-essentials.zip
   용량: 100 MB
   설치 시간: 5분
   ⚠️  중요: 압축 해제 후 bin 폴더를 시스템 PATH에 추가

🔧 NVIDIA CUDA Toolkit (선택 - GPU 가속)
   다운로드: https://developer.nvidia.com/cuda-downloads
   용량: 3.5 GB
   설치 시간: 10분
   조건: NVIDIA GPU (RTX 20/30/40 시리즈) 필요
   효과: 쇼츠 생성 속도 4-8배 향상


📥 2단계: AI 쇼츠 프로그램 다운로드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

방법 1: GitHub ZIP 다운로드 (가장 쉬움 - 권장)
   1. https://github.com/hompystory-coder/azamans 접속
   2. 녹색 "Code" 버튼 클릭
   3. "Download ZIP" 클릭
   4. 압축 해제 (우클릭 → 압축 풀기)
   5. azamans-main/local-shorts-system 폴더 열기

방법 2: Git Clone (Git 설치 후)
   터미널(CMD)에서 실행:
   git clone https://github.com/hompystory-coder/azamans.git
   cd azamans/local-shorts-system

방법 3: 압축 파일 다운로드
   파일: local-shorts-installer.tar.gz
   위치: GitHub 저장소 루트
   용량: 28 KB (AI 모델 제외)


🚀 3단계: 설치 및 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. local-shorts-system 폴더 열기

2. install_windows.bat 더블클릭 (5분)
   → Python 라이브러리 자동 설치

3. download_models_windows.bat 더블클릭 (30-60분, 최초 1회만)
   → AI 모델 자동 다운로드 (15.3 GB)
   - Stable Diffusion XL (6.9 GB)
   - AnimateDiff (1.7 GB)
   - Coqui TTS (2.0 GB)
   - LLaMA 3.1 (4.7 GB)

4. start_gui_windows.bat 더블클릭 (2분)
   → GUI 프로그램 실행

5. URL 입력 → 캐릭터 선택 → Generate 클릭!
   → 7-8분 후 output 폴더에서 쇼츠 확인

❓ 자주 묻는 질문 (FAQ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 인터넷 없이 사용 가능한가요?
A: 네! 최초 설치 및 AI 모델 다운로드(15.3GB)만 인터넷이 필요합니다.
   이후 완전 오프라인 사용 가능합니다.

Q: GPU가 없어도 되나요?
A: 가능하지만 속도가 느립니다:
   - GPU 있음 (RTX 3060): 7-8분/쇼츠
   - CPU만: 30-60분/쇼츠

Q: 설치가 어렵나요?
A: 전혀요! BAT 파일을 더블클릭만 하면 됩니다:
   1. install_windows.bat (설치)
   2. download_models_windows.bat (모델 다운로드)
   3. start_gui_windows.bat (실행)

Q: 어떤 쇼츠를 만들 수 있나요?
A: - 블로그/뉴스 기사 → 쇼츠
   - 텍스트 내용 → 쇼츠
   - 39종 캐릭터 지원
   - 16:9, 9:16, 1:1 화면 비율

Q: 비용이 얼마나 절약되나요?
A: OpenAI/ElevenLabs API 대비:
   - 월 100개 생성 시: $63/월 → $0/월
   - 연간 절약: $756
   - 비용 절감률: 92-95%
