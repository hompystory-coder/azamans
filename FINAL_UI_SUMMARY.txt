🚀 사용자 액션

  1. 브라우저에서 https://ai-shorts.neuralgrid.kr/ 접속
  2. "지금 시작하기" 버튼 클릭 (또는 헤더의 "새 프로젝트")
  3. 프로젝트 제목 입력
  4. 캐릭터 선택 (10개 프리셋 중)
  5. AI 음성 선택 (8개 중)
  6. 블로그 URL 입력
  7. 자동 생성 시작!
