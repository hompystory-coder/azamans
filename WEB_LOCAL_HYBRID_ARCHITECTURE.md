# Web-Based Local AI Shorts Generator
# 웹 기반 + 로컬 리소스 하이브리드 아키텍처

## 🎯 핵심 아이디어

사용자 경험:
1. 웹사이트 접속 (예: shorts.ai)
2. "시작하기" 버튼 클릭
3. 브라우저가 AI 모델 자동 다운로드 (한 번만)
4. 사용자 GPU로 쇼츠 생성
5. 모든 처리가 브라우저 안에서!

---

## 🏗️ 기술 아키텍처

### Frontend (웹 인터페이스)
```
Next.js / React
- 아름다운 UI
- 간단한 사용법
- 진행 상황 표시
```

### AI 실행 (브라우저 안에서)
```
WebGPU + ONNX Runtime Web
- Stable Diffusion (이미지)
- AnimateDiff (비디오)
- Whisper/TTS (음성)
- LLaMA (스크립트)

모두 사용자 GPU에서 실행!
```

### 저장소 (로컬)
```
IndexedDB (브라우저 저장소)
- 생성된 쇼츠
- 캐시된 모델
- 사용자 설정
```

---

## 📦 모델 최적화

### 문제: 원본 모델이 너무 큼
```
Stable Diffusion XL: 6.9GB
AnimateDiff: 1.7GB
Coqui TTS: 2.0GB
LLaMA 3.1: 4.7GB
---
총: 15.3GB (너무 큼!)
```

### 해결: 경량화 모델 사용
```
✅ Stable Diffusion v1.5 ONNX: 2GB
✅ AnimateDiff Lite: 500MB
✅ TTS Lightweight: 100MB
✅ LLaMA 3.2 1B: 1GB
---
총: 3.6GB (훨씬 작음!)

추가 최적화:
- INT8 양자화 (50% 크기 감소)
- Progressive loading (점진적 다운로드)
- Web Worker (백그라운드 실행)
```

## 🎯 경쟁 우위

### vs 클라우드 SaaS
```
우리:
✅ 무료 (무제한)
✅ 프라이버시
✅ 빠름 (로컬 GPU)
✅ 오프라인 가능

그들:
✅ 설치 쉬움
❌ 비용 ($19-99/월)
❌ 데이터 업로드
❌ 느림 (대기열)
```

### vs 로컬 설치 프로그램
```
우리:
✅ 설치 없음 (웹)
✅ 자동 업데이트
✅ 쉬운 사용

그들:
✅ 완전 오프라인
❌ 설치 복잡
❌ 업데이트 수동
```

→ **우리가 최고의 두 세계!**
