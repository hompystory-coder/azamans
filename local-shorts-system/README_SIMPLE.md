# AI Shorts Generator - Quick Start

## Super Easy Installation! (2 Steps Only)

### Step 1: Install (First time only - 40-70 min)

**Double-click one of these files:**

- `EASY_INSTALLER.bat` (English version)
- `INSTALL_KOREAN.bat` (Korean-friendly version)

What happens?
```
1. Check Python, Ollama, FFmpeg
2. Install Python libraries (5 min)
3. Download AI models - 15.3GB (30-60 min)

```

### Step 2: Run (Every time - 2 seconds)

**Double-click:**
- `START_HERE.bat`

## Download Links

If you don't have required programs:

1. **Python 3.10**: https://www.python.org/downloads/windows/
   - IMPORTANT: Check "Add Python to PATH" during installation!

2. **Ollama**: https://ollama.ai/download/windows
   - For running LLaMA 3.1 AI model

3. **FFmpeg**: https://www.gyan.dev/ffmpeg/builds/
   - Download: ffmpeg-release-essentials.zip
   - Extract to: C:\ffmpeg
   - Add to PATH: C:\ffmpeg\bin

## Troubleshooting

### Problem: "Python not found"
**Solution:**
1. Install Python from link above
2. Check "Add Python to PATH" during installation
3. Restart computer
4. Run installer again

### Problem: "Ollama not found"
**Solution:**
1. Install Ollama from link above
2. Ollama will start automatically
3. Run installer again

### Problem: "FFmpeg not found"
**Solution:**
1. Download FFmpeg
2. Extract to C:\ffmpeg
3. Add C:\ffmpeg\bin to system PATH
4. Restart computer
5. Run installer again

### Problem: Download is slow
**This is normal!**
- Total size: 15.3GB
- Can take 30-60 minutes
- Just wait, it will complete
