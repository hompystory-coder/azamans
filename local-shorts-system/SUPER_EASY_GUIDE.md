## ⚠️ 필수 프로그램 자동 설치 안내

`EASY_INSTALLER.bat`가 프로그램이 없으면 자동으로 알려줍니다:

### Python이 없으면:
```
❌ Python이 설치되지 않았습니다!

📥 Python 다운로드 페이지를 엽니다...
(브라우저가 자동으로 열립니다)

⚠️ 중요:
   1. "Python 3.10.11" 다운로드
   2. 설치 시 "Add Python to PATH" 체크!
   3. 설치 후 이 프로그램을 다시 실행하세요
```

### Ollama가 없으면:
```
❌ Ollama가 설치되지 않았습니다!

📥 Ollama 다운로드 페이지를 엽니다...
(브라우저가 자동으로 열립니다)

⚠️ 설치 후 이 프로그램을 다시 실행하세요
```

### FFmpeg가 없으면:
```
❌ FFmpeg가 설치되지 않았습니다!

📥 FFmpeg 다운로드 페이지를 엽니다...

⚠️ 설치 방법:
   1. "ffmpeg-release-essentials.zip" 다운로드
   2. C:\ffmpeg 에 압축 해제
   3. 시스템 환경 변수 PATH에 "C:\ffmpeg\bin" 추가
   4. 컴퓨터 재시작
```
