
```
특허출원 메뉴
→ 전자출원
→ 서지사항 입력 (위 서식 참조)
→ 명세서 첨부 (PATENT_APPLICATION.md 내용 복사)
→ 요약서 입력 (위 요약서 복사)
→ 도면 첨부 (7개)
→ 감면 신청 정보 연결
→ 제출!
```

### Step 4: 출원번호 확보
```
제출 완료 후 출원번호 즉시 발급
예: 10-2024-XXXXXXX
```
