# 특허 출원 명세서 (초안)

---

## 【서지사항】

### 【발명의 명칭】
**브라우저 기반 인공지능 자동화 단편 영상 생성 시스템 및 방법**  
(AI-Powered Automated Short-Form Video Generation System and Method Based on Web Browser)

### 【기술분야】
본 발명은 인공지능 기술을 활용한 영상 콘텐츠 생성 시스템에 관한 것으로, 더욱 상세하게는 사용자의 간단한 텍스트 입력만으로 인공지능이 자동으로 프롬프트를 확장하고, 최적의 시각적 스타일을 선택하며, 이미지 생성, 음성 합성, 비디오 편집을 수행하여 전문가 수준의 단편 영상(쇼츠)을 웹 브라우저 환경에서 완전 자동으로 생성하는 시스템 및 방법에 관한 것이다.

---

## 【배경기술】

### 1. 종래 기술의 문제점

#### 1.1 전문 지식 요구
종래의 영상 콘텐츠 제작 시스템은 사용자가 영상 편집 소프트웨어의 사용법, 색상 이론, 구도, 조명 등 전문적인 지식을 보유해야만 고품질의 콘텐츠를 제작할 수 있었다. 이는 일반 사용자의 진입 장벽을 높이는 주요 요인이었다.

#### 1.2 복잡한 프롬프트 작성
최근 생성형 AI를 활용한 이미지/영상 생성 서비스들이 등장하였으나, 고품질의 결과물을 얻기 위해서는 사용자가 매우 상세하고 전문적인 프롬프트(예: "4K resolution, cinematic lighting, rule of thirds, bokeh background, professional color grading" 등)를 작성해야 했다. 일반 사용자는 이러한 전문 용어를 모르거나 적절히 조합하는 방법을 알지 못하여 만족스러운 결과를 얻기 어려웠다.

#### 1.3 수동적인 스타일 선택
기존 시스템에서는 사용자가 애니메이션, 사실적, 사이버펑크 등 다양한 시각적 스타일 중 하나를 직접 선택해야 했다. 그러나 대부분의 사용자는 자신의 콘텐츠에 어떤 스타일이 적합한지 판단하기 어려워, 시행착오를 반복하거나 최적이 아닌 스타일을 선택하게 되었다.

#### 1.4 소프트웨어 설치 및 하드웨어 제약
종래의 AI 영상 생성 도구들은 Python, CUDA, 대용량 AI 모델 등을 로컬 환경에 설치해야 했으며, 이는 일반 사용자에게 큰 부담이었다. 또한 고성능 GPU를 필요로 하여 하드웨어 접근성이 낮았다.

#### 1.5 단계별 수동 작업
이미지 생성, 음성 합성, 비디오 편집 등 각 단계가 분리되어 있어, 사용자가 각 단계를 수동으로 진행하고 중간 결과물을 다음 단계로 전달해야 했다. 이는 전체 제작 과정을 복잡하고 시간 소모적으로 만들었다.

## 【발명의 내용】

### 【해결하고자 하는 과제】

본 발명은 상기와 같은 종래 기술의 문제점을 해결하기 위한 것으로, 그 목적은 다음과 같다:

1. **간단한 입력의 지능형 확장**: 사용자가 "고양이"와 같은 단순한 단어만 입력해도, 인공지능이 자동으로 시각적 디테일, 조명, 색상, 구도 등 전문적인 요소를 추가하여 고품질 프롬프트로 확장하는 시스템을 제공한다.

2. **컨텍스트 기반 자동 스타일 선택**: 입력 내용의 키워드, 분위기, 콘텐츠 타입을 분석하여 최적의 시각적 스타일을 자동으로 선택하는 시스템을 제공한다.

3. **완전 자동화 파이프라인**: 프롬프트 확장 → 스타일 선택 → 이미지 생성(3장면) → 음성 합성 → 비디오 편집의 전 과정을 사용자 개입 없이 자동으로 수행하는 시스템을 제공한다.

4. **Zero-Install 웹 기반 실행**: 소프트웨어 설치 없이 웹 브라우저만으로 실행 가능하며, WebGPU를 활용하여 사용자의 로컬 GPU를 활용하는 시스템을 제공한다.

5. **접근성 향상**: 전문 지식이 없는 초보자도 프로 수준의 단편 영상을 5분 이내에 제작할 수 있는 시스템을 제공한다.

## 【과제의 해결 수단】

상기 목적을 달성하기 위한 본 발명의 일 실시예에 따른 브라우저 기반 인공지능 자동화 단편 영상 생성 시스템은:

### (1) **지능형 프롬프트 확장 모듈** (AI Prompt Enhancer)

사용자의 간단한 텍스트 입력을 수신하여, 다음의 하위 모듈을 통해 확장된 프롬프트를 생성한다:

- **키워드 분석 모듈**: 입력 텍스트에서 주요 키워드를 추출하고, 사전 정의된 키워드 데이터베이스(동물, 자연, 음식, 기술, 감정, 행동 등 6개 이상의 카테고리)와 매칭하여 콘텐츠 유형을 분류한다.

- **분위기 감지 모듈**: 입력 텍스트 및 추출된 키워드를 기반으로 콘텐츠의 분위기(cheerful, mysterious, peaceful, intense, nostalgic, funny 등)를 자동 판단한다.

- **시각적 디테일 추가 모듈**: 분석 결과를 바탕으로 조명(lighting), 분위기(atmosphere), 색상(colors), 구도(composition), 품질(quality) 등 5가지 이상의 카테고리에서 시각적 디테일을 자동으로 선택하여 프롬프트에 추가한다.

- **다중 장면 생성 모듈**: 확장된 프롬프트를 기반으로 서로 다른 구도와 의미를 가진 3개 이상의 장면(예: 설정 장면, 액션 장면, 클로즈업 장면)을 자동 생성한다.

- **네거티브 프롬프트 생성 모듈**: 콘텐츠 타입에 부적합한 요소(예: 자연 콘텐츠의 경우 "urban, artificial")를 자동으로 제외 항목으로 지정한다.

### (2) **지능형 스타일 선택 모듈** (Smart Style Selector)

확장된 프롬프트 및 분석 정보를 기반으로 최적의 시각적 스타일을 자동 선택한다:

- **다차원 점수 시스템**: 키워드 매칭(가중치 3점), 분위기 매칭(가중치 5점), 콘텐츠 타입 매칭(가중치 4점) 등 다수의 기준으로 각 스타일 프리셋의 적합성 점수를 산출한다.

- **스타일 프리셋 데이터베이스**: 애니메이션, 사이버펑크, 판타지, 미니멀리스트, 빈티지, 자연 다큐멘터리, 호러, 코미디, 교육용, 동기부여 등 10개 이상의 사전 정의된 스타일 프리셋을 포함한다. 각 프리셋은 promptModifier, negativePrompt, imageStyle(색상, 분위기, 시각 스타일), videoSettings(트랜지션, 지속시간, FPS), audioSettings(음성 스타일, 속도, 음높이) 등의 매개변수를 포함한다.

- **플랫폼 최적화 모듈**: 콘텐츠 타입 및 복잡도를 기반으로 YouTube Shorts, TikTok, Instagram Reels, Square(1:1) 등 타겟 플랫폼을 자동 선택하고, 해당 플랫폼에 최적화된 해상도, 화면비, 권장 길이를 설정한다.

- **음악/음성 매칭 모듈**: 선택된 스타일 및 분위기에 적합한 배경 음악 유형(upbeat-pop, ambient-dark, lo-fi-chill 등) 및 음성 스타일을 자동 선택한다.

### (3) **브라우저 기반 이미지 생성 모듈**

WebGPU 및 ONNX Runtime Web을 활용하여 사용자의 로컬 GPU에서 이미지 생성을 수행한다:

- **GPU 감지 및 초기화**: WebGPU API를 통해 사용자 기기의 GPU를 감지하고, 사용 가능한 경우 고성능 GPU 어댑터를 초기화한다. GPU를 사용할 수 없는 경우 WebAssembly(WASM) 백엔드로 자동 폴백한다.

- **AI 모델 로딩 및 캐싱**: Stable Diffusion 등의 AI 모델을 ONNX 형식으로 로드하고, IndexedDB를 활용하여 브라우저 내에 캐싱함으로써 재사용 시 로딩 시간을 단축한다.

- **프리셋 적용 이미지 생성**: 선택된 스타일 프리셋의 매개변수를 적용하여 확장된 프롬프트로부터 다중 장면(3장 이상)의 이미지를 순차적으로 생성한다. 생성 실패 시 Hugging Face API 등 클라우드 기반 백업 시스템으로 자동 전환한다.

### (4) **자동 음성 합성 모듈**

- **스크립트 자동 생성**: 확장된 프롬프트 및 선택된 스타일을 기반으로 영상에 적합한 내레이션 스크립트를 자동 생성한다.

- **TTS(Text-to-Speech) 합성**: Web Speech API 또는 외부 TTS 서비스를 활용하여, 선택된 음성 스타일(cheerful, dramatic, calm 등), 속도(rate), 음높이(pitch) 설정에 따라 스크립트를 음성으로 합성한다.

### (5) **자동 비디오 편집 및 합성 모듈**

- **FFmpeg.wasm 기반 편집**: FFmpeg의 WebAssembly 버전을 활용하여 브라우저 내에서 생성된 이미지들과 음성을 결합하고, 선택된 트랜지션 효과(fade, slide, zoom, dissolve)를 적용한다.

- **플랫폼별 최적화**: 선택된 플랫폼의 해상도(예: 1080x1920 for 9:16) 및 FPS 설정에 맞춰 최종 비디오를 렌더링한다.

- **자동 저장 및 관리**: 완성된 비디오를 로컬 스토리지(LocalStorage/IndexedDB)에 자동 저장하고, 갤러리 UI를 통해 사용자가 이전 생성물을 관리할 수 있도록 한다.

### (6) **사용자 인터페이스 모듈**

- **원클릭 입력 UI**: 사용자가 단일 텍스트 입력 필드에 간단한 단어나 문장을 입력하고 하나의 버튼을 클릭하는 것만으로 전체 프로세스를 시작할 수 있도록 한다.

- **실시간 진행 상황 표시**: 6단계 이상의 생성 파이프라인(AI 분석 → 프롬프트 확장 → 스타일 선택 → 이미지 생성 → 음성 생성 → 비디오 합성) 각각의 진행률과 상태(대기/처리중/완료/오류)를 실시간으로 시각화한다.

- **예시 입력 버튼**: 사용자의 이해를 돕기 위해 "고양이", "우주 탐험", "맛있는 음식" 등 다양한 카테고리의 예시 입력을 버튼으로 제공한다.

## 【발명의 효과】

본 발명에 따른 브라우저 기반 인공지능 자동화 단편 영상 생성 시스템은 다음과 같은 효과를 갖는다:

### 1. 접근성 향상
전문 지식이 없는 일반 사용자도 간단한 단어 입력만으로 전문가 수준의 단편 영상을 제작할 수 있어, 콘텐츠 제작의 민주화를 실현한다.

### 2. 프롬프트 작성 부담 해소
사용자가 복잡한 AI 프롬프트 작성법을 학습할 필요 없이, 시스템이 자동으로 전문적인 시각적 디테일을 추가하여 고품질 결과물을 보장한다.

### 3. 최적 스타일 자동 선택
콘텐츠의 키워드, 분위기, 타입을 종합적으로 분석하여 가장 적합한 시각적 스타일을 자동으로 선택함으로써, 사용자의 시행착오를 최소화하고 결과물의 품질을 향상시킨다.

### 4. 시간 및 비용 절감
종래의 수동 제작 방식 대비 제작 시간을 5분 이내로 단축하며, 소프트웨어 구입 비용 및 고성능 하드웨어 투자 없이 무료로 사용 가능하다.

### 5. Zero-Install 편의성
웹 브라우저만으로 실행 가능하여 설치 과정이 불필요하며, 어떤 플랫폼(Windows, Mac, Linux, 모바일)에서도 동일하게 사용할 수 있다.

### 6. 프라이버시 보호
WebGPU를 활용한 로컬 처리 방식으로 사용자의 콘텐츠가 외부 서버로 전송되지 않아 프라이버시를 보호한다.

### 7. 확장성 및 재사용성
모듈화된 구조로 인해 새로운 스타일 프리셋, AI 모델, 플랫폼 템플릿을 쉽게 추가할 수 있으며, 각 모듈을 독립적으로 업그레이드할 수 있다.

## 【청구항】

### 【청구항 1】 (독립항 - 시스템)

사용자의 간단한 텍스트 입력으로부터 완성된 단편 영상을 자동 생성하는 브라우저 기반 인공지능 자동화 단편 영상 생성 시스템에 있어서,

사용자로부터 수신한 텍스트 입력을 분석하여 키워드를 추출하고, 상기 키워드를 기반으로 콘텐츠의 분위기 및 타입을 자동 판단하며, 시각적 디테일을 추가하여 확장된 프롬프트 및 복수의 장면 프롬프트를 생성하는 **지능형 프롬프트 확장 모듈**;

상기 확장된 프롬프트 및 분석 정보를 기반으로, 사전 정의된 복수의 스타일 프리셋 각각에 대해 키워드 매칭, 분위기 매칭, 콘텐츠 타입 매칭의 가중 점수를 산출하고, 최고 점수의 스타일 프리셋을 자동 선택하며, 타겟 플랫폼 및 음악/음성 설정을 결정하는 **지능형 스타일 선택 모듈**;

웹 브라우저 환경에서 WebGPU API를 통해 로컬 GPU를 활용하여 ONNX 형식의 AI 모델을 실행하고, 상기 선택된 스타일 프리셋의 매개변수를 적용하여 상기 복수의 장면 프롬프트로부터 복수의 이미지를 생성하는 **브라우저 기반 이미지 생성 모듈**;

상기 확장된 프롬프트 및 선택된 스타일을 기반으로 내레이션 스크립트를 자동 생성하고, 상기 스타일 프리셋에 정의된 음성 스타일, 속도, 음높이 설정에 따라 음성을 합성하는 **자동 음성 합성 모듈**; 및

FFmpeg의 WebAssembly 버전을 활용하여 상기 생성된 복수의 이미지와 상기 합성된 음성을 결합하고, 상기 스타일 프리셋에 정의된 트랜지션 효과를 적용하여 최종 비디오를 렌더링하는 **자동 비디오 편집 및 합성 모듈**;

### 【실시예 1: 시스템 구성】

도1은 본 발명의 일 실시예에 따른 브라우저 기반 인공지능 자동화 단편 영상 생성 시스템(100)의 전체 구성도이다.

#### 1.1 지능형 프롬프트 확장 모듈(110)
