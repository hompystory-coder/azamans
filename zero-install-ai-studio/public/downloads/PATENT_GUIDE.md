### 【도3】 스타일 선택 점수 산출 과정
```
┌─────────────────────────────────────────────────────────────┐
│         지능형 스타일 선택 모듈 - 점수 산출 과정             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  입력 분석 정보:                                            │
│  • keywords: ['고양이', 'cat', 'pet']                      │
│  • mood: 'cheerful'                                         │
│  • contentType: 'nature'                                    │
│  • targetAudience: 'general'                                │
│         ↓                                                    │
│  ┌─────────────────────────────────────────────────────┐  │
│  │  스타일 프리셋 순회                                  │  │
│  │  for each preset in [anime-girl, cyberpunk, ...]    │  │
│  └──────────────┬──────────────────────────────────────┘  │
│                 ↓                                           │
│  ┌─────────────────────────────────────────────────────┐  │
│  │  점수 계산                                           │  │
│  │                                                      │  │
│  │  【anime-girl 프리셋】                               │  │
│  │  ┌───────────────────────────────────────────────┐  │  │
│  │  │ 1. 키워드 매칭 점수                           │  │  │
│  │  │    입력: ['고양이', 'cat']                    │  │  │
│  │  │    프리셋 키워드: ['고양이', '강아지', ...]   │  │  │
│  │  │    매칭: '고양이' → +3점                      │  │  │
│  │  └───────────────────────────────────────────────┘  │  │
│  │  ┌───────────────────────────────────────────────┐  │  │
│  │  │ 2. 분위기 매칭 점수                           │  │  │
│  │  │    입력 mood: 'cheerful'                      │  │  │
│  │  │    프리셋 mood: ['cheerful', 'funny']        │  │  │
│  │  │    매칭: 'cheerful' → +5점                    │  │  │
│  │  └───────────────────────────────────────────────┘  │  │
│  │  ┌───────────────────────────────────────────────┐  │  │
│  │  │ 3. 콘텐츠 타입 매칭 점수                      │  │  │
│  │  │    입력 type: 'nature'                        │  │  │
│  │  │    프리셋 type: ['character', 'nature']       │  │  │
│  │  │    매칭: 'nature' → +4점                      │  │  │
│  │  └───────────────────────────────────────────────┘  │  │
│  │  ┌───────────────────────────────────────────────┐  │  │
│  │  │ 4. 오디언스 조정                              │  │  │
│  │  │    target: 'general' → 추가 점수 없음         │  │  │
│  │  └───────────────────────────────────────────────┘  │  │
│  │                                                      │  │
│  │  ✅ 총점: 3 + 5 + 4 = **12점**                       │  │
│  └─────────────────────────────────────────────────────┘  │
│                 ↓                                           │
│  ┌─────────────────────────────────────────────────────┐  │
│  │  다른 프리셋 점수 비교                               │  │
│  │  • anime-girl: 12점 ⭐ (최고점)                      │  │
│  │  • nature: 9점                                       │  │
│  │  • comedy: 5점                                       │  │
│  │  • horror: 0점                                       │  │
│  └──────────────┬──────────────────────────────────────┘  │
│                 ↓                                           │
│  ┌─────────────────────────────────────────────────────┐  │
│  │  최종 선택                                           │  │
│  │  • primary: anime-girl                               │  │
│  │  • alternatives: [nature, comedy]                    │  │
│  │  • confidence: 0.85 (12/14 정규화)                  │  │
│  │  • reason: "키워드, 분위기, 타입 매칭"               │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**설명**: 다차원 점수 시스템을 통한 스타일 자동 선택 과정
