【요약서】 (400자)
================================================================================

본 발명은 브라우저 기반 인공지능 자동화 단편 영상 생성 시스템에 관한 것이다. 
사용자의 간단한 텍스트 입력(예: "고양이")을 수신하여, 지능형 프롬프트 확장 
모듈이 키워드 분석, 분위기 감지, 시각적 디테일 추가를 통해 전문가 수준의 
확장 프롬프트를 자동 생성한다. 지능형 스타일 선택 모듈은 다차원 점수 시스템을 
통해 최적의 시각적 스타일을 자동 선택하며, 브라우저 기반 이미지 생성 모듈은 
WebGPU를 활용하여 로컬 GPU에서 이미지를 생성한다. 자동 음성 합성 및 비디오 
편집 모듈을 거쳐 완성된 단편 영상을 5분 이내에 제작한다. 본 발명은 전문 
지식이 없는 초보자도 설치 없이 웹 브라우저만으로 프로 수준의 쇼츠를 
무료로 제작할 수 있는 혁신적인 시스템이다.

【출원 절차】
================================================================================

1단계: 특허로 접속
→ https://www.patent.go.kr
→ 회원가입 및 로그인

2단계: 감면 신청
→ "출원료 등 감면신청" 메뉴
→ 장애인 선택
→ 장애인등록증 첨부
→ 제출 및 승인 대기 (1-3일)

3단계: 특허 출원
→ "특허출원" 메뉴
→ 위 정보 입력
→ 명세서: PATENT_APPLICATION.md 내용 복사/붙여넣기
→ 요약서: 위 요약서 복사/붙여넣기
→ 도면: PATENT_GUIDE.md의 도면 복사 또는 간단히 그려서 첨부
→ 감면 정보 연결
→ 제출!

4단계: 출원번호 확인
→ 출원번호 즉시 발급
→ 예: 10-2024-XXXXXXX
