## 🚀 향후 확장 가능성

### 1️⃣ 나레이션 자동 생성 (AI)
**현재**: 75개 고정 나레이션 풀  
**개선**: Ollama가 주제별 커스텀 나레이션 자동 생성

```python
# 예시: Ollama 활용 나레이션 생성
def generate_narration_with_ollama(scene_title, mood, act_name):
    prompt = f"""
    다음 장면에 맞는 30자 이내의 궁금증 유발 나레이션을 작성해줘:
    - 장면: {scene_title}
    - 분위기: {mood}
    - 막: {act_name}
    
    예시: "여러분, 이건 정말 믿기 힘든 이야기인데 한번 들어보세요."
    """
    narration = ollama_api.generate(model="llama3.1:8b", prompt=prompt)
    return narration
```

**장점**:
- ✅ 무한히 다양한 나레이션
- ✅ 주제별 맞춤 표현
- ✅ 중복 완전 제거

**단점**:
- ⚠️ 응답 속도 느림 (10-30초)
- ⚠️ 품질 검증 필요

### 2️⃣ 장르별 스타일 자동 적용
**현재**: 모든 스토리가 동일한 5막 구조  
**개선**: 장르 자동 인식 → 맞춤 구조

```python
# 예시: 장르 인식 및 구조 선택
def detect_genre_and_apply_structure(user_input):
    prompt = f"""
    다음 스토리의 장르를 판단하고, 적절한 구조를 추천해줘:
    "{user_input}"
    
    장르 옵션: 동화, 액션, 로맨스, 공포, SF, 코미디
    구조 옵션: 3막, 5막, 영웅의 여정
    
    JSON 형식으로 응답:
    {{"genre": "동화", "structure": "5막", "tone": "따뜻한"}}
    """
    result = ollama_api.generate(model="llama3.1:8b", prompt=prompt)
    return json.loads(result)
```

### 3️⃣ 멀티모달 확장 (이미지 분석)
**현재**: 텍스트 → 이미지 생성만 가능  
**개선**: 사용자가 이미지 업로드 → Ollama가 분석

**요구사항**:
- ❌ Ollama 자체는 이미지 분석 불가
- ✅ LLaVA (Vision + Language) 모델 추가 설치 필요

```bash
# LLaVA 설치 (멀티모달 모델)
ollama pull llava:7b
```

**활용 사례**:
```python
# 이미지 업로드 → 스토리 자동 생성
def generate_story_from_image(image_path):
    prompt = "이 이미지를 보고 30초 쇼츠 스토리를 만들어줘"
    story = ollama_api.generate(
        model="llava:7b", 
        prompt=prompt,
        images=[image_path]
    )
    return story
```

### 4️⃣ 다국어 쇼츠 생성
**현재**: 한국어 전용

```python
# 예시: 다국어 나레이션 생성
def generate_multilang_narration(scene, language="en"):

    Translate this scene narration to {language}:
    "{scene['narration']}"
    
    Keep the same engaging tone and curiosity-inducing style.

    return translated
```

| 항목 | Ollama (로컬) | 외부 API (OpenAI 등) |
|------|--------------|---------------------|

| **프라이버시** | ✅ 완벽 (로컬) | ❌ 데이터 전송 |
