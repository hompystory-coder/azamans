# AI Shorts Generator - Product Strategy
# 목표: 판매하면서 사용자 행복도 최대화

## 🎯 비즈니스 모델

### 3단계 전략

#### Phase 1: MVP (최소 실행 제품) - 2주
```
목표: 빠르게 시장 검증

제품 형태:
- 웹 기반 SaaS (설치 불필요)
- 클라우드 API 사용
- 간단한 UI
- "3분 안에 첫 쇼츠 생성"

가격:
- 무료 플랜: 월 5개 쇼츠
- 스타터: $19/월 (50개)
- 프로: $49/월 (무제한)

기술 스택:
- Frontend: Next.js (빠른 웹앱)
- Backend: FastAPI (Python)
- AI: Replicate API (GPU 서버 대여)
- 배포: Vercel + Railway

비용:
- 개발: $0 (이미 코드 있음)
- 서버: $50-100/월
- 무료 유저 1명당: $0.50
- 유료 유저 1명당 수익: $19-49

손익분기점:
- 유료 고객 5명이면 수익 발생
```

#### Phase 2: 프리미엄 기능 - 1개월
```
추가 기능:
- 더 많은 캐릭터 (100종+)
- 더 긴 영상 (최대 3분)
- 브랜드 워터마크 제거
- API 접근
- 우선 처리 (더 빠른 생성)

가격:
- 무료: 5개/월 (워터마크 있음)
- 스타터 $19: 50개/월
- 프로 $49: 200개/월
- 비즈니스 $149: 무제한 + API

고객 행복도:
✅ 무료로도 사용 가능 (체험)
✅ 설치 없음 (웹에서 즉시)
✅ 결과가 빠름 (3-5분)
✅ 품질 좋음 (프로 AI 사용)
```

#### Phase 3: 하이브리드 옵션 - 2개월
```
고급 사용자를 위한 선택:
- 클라우드 (쉽고 빠름)
- 온프레미스 (프라이버시, 무제한)

온프레미스 라이선스:
- 엔터프라이즈: $999 일회성
- 자체 서버에 설치
- 무제한 생성
- 데이터 외부 유출 없음
- 1년 지원 포함

타겟:
- 에이전시
- 큰 유튜버
- 기업 마케팅팀
```

## 🚀 즉시 실행 가능한 계획

### Week 1-2: 웹 버전 만들기

**기술적 변경사항:**

1. **Backend API 분리**
   ```python
   현재: 로컬 PC에서 모든 AI 실행
   변경: API 서버에서 실행
   
   /api/shorts/generate
   - 입력: URL + 캐릭터
   - 출력: 쇼츠 비디오 URL
   ```

2. **Frontend 웹앱**
   ```javascript
   - Next.js로 간단한 웹사이트
   - 로그인 (이메일/구글)
   - 크레딧 시스템
   - 비디오 갤러리
   ```

3. **클라우드 AI 사용**
   ```
   현재: Stable Diffusion 로컬
   변경: Replicate API (GPU 대여)
   
   장점:
   - 설치 불필요
   - 항상 빠름
   - 자동 스케일링
   
   비용:
   - 이미지 생성: $0.0023/장
   - 비디오 생성: $0.05/영상
   - 음성 생성: $0.006/분
   - 쇼츠 1개당: ~$0.15
   ```

4. **결제 연동**
   ```
   - Stripe (신용카드)
   - 한국: 토스페이먼츠
   - 자동 구독 관리
   ```

## 😊 고객 행복도 극대화

### 1. 무료 체험 (진입 장벽 제거)
```

```

- 지속적 업데이트
- 새로운 캐릭터 추가
- 새로운 기능 추가
```

### 4. 간편함
```
❌ Python 설치
❌ GPU 필요
❌ 15GB 다운로드
❌ 복잡한 설정

✅ 웹사이트 접속
✅ 클릭 몇 번
✅ 완성!
```

### 5. 투명한 가격
```
