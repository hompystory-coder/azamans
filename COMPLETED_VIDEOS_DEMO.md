1. 브라우저에서 https://mfx.neuralgrid.kr 접속
2. 상단 메뉴 또는 `/generation` 페이지로 이동
3. 완료된 비디오 목록에서 재생 버튼 클릭

### 방법 2: 직접 URL로 비디오 다운로드/시청
위의 각 비디오 URL을 브라우저 주소창에 복사하여 붙여넣기

예시:
```

```

### 방법 3: 다운로드 링크로 저장
URL을 우클릭 → "다른 이름으로 링크 저장"
