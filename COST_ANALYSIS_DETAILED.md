   - JSON 포맷 자동 생성

3. **이미지 자동 매칭**
   - 크롤링한 이미지 자동 매핑
   - 장면별 최적 이미지 배치

4. **안정적인 처리**
   - Gemini API 장애 없음
   - 100% 로컬 처리
   - 즉시 응답 (API 호출 대기 없음)

### 코드 예시 (backend/src/routes/script.js):
```javascript
// 콘텐츠를 문장으로 분할
const sentences = content
  .split(/[.!?]\s+/)
  .map(s => s.trim())
  .filter(s => s.length >= 15 && s.length <= 150)
  .slice(0, sceneCount * 2);

// 적절한 길이의 문장 선택 (15-50자)
const selectedSentences = sentences
  .filter(s => s.length <= 50)
  .slice(0, sceneCount);

// JSON 응답 구조 생성
const scenes = selectedSentences.map((sentence, index) => ({
  sceneNumber: index + 1,
  narration: sentence,
  imageDescription: `장면 ${index + 1}`,
  duration: Math.min(Math.max(Math.ceil(sentence.length / 15), 3), 6)
}));
```
